# fyp_darksight

## Overview
An offensive data preprocessing tool designed to poison and disrupt Vision-Language Models (VLMs) using Nightshade. This repository was created as part of my final year project on investigating data poisoning techniques on text-to-image models.

## Organisation of Repository

```
fyp_darksight/
├── splits/                    # Contains train/validation/test dataset splits
├── test_set/                  # The test dataset used to evaluate the performance of the VLM
├── to_be_trained_on_VLM/      # The dataset used to finetune the target VLM
├── cap.json                   # Caption annotation file of to_be_trained_on_VLM/
├── descriptions_clean.json    # Text descriptions generated by the clean VLM
├── descriptions_poisoned.json # Text descriptions generated by the poisoned VLM
├── evaluate_VLM.py            # Performance evaluation script
├── evaluation_results.json    # Output JSON evaluation file
├── output_clean_VLM.py        # Generate descriptions from clean model
├── output_poisoned_VLM.py     # Generate descriptions from poisoned model
├── prepare_split.py           # Dataset preparation and splitting script
├── process_results.py         # Parse output JSON file
├── README.md                  # This file
├── requirements.txt           # Dependencies to install
├── train_qwen2vl_lora.py      # VLM fine-tuning with LoRA
└── VLM_dataset_conversion.py  # Convert the dataset to be suitable as input to train the VLM 
```

## Setup
Begin by installing all required dependencies:
```bash
pip install -r requirements.txt
```

**Important Note:** PyTorch must be installed separately to match your system configuration. Visit the [official PyTorch installation guide](https://pytorch.org/get-started/locally/) and select the appropriate version for your operating system and hardware.

## Test Dataset Information
This project utilizes the [512 x 512 Dogs Images dataset from Kaggle](https://www.kaggle.com/datasets/greg115/dogs-big) as the test dataset for evaluating model performance.

## Usage

### Step 1: Poison the Dataset with Nightshade
Process your source dataset through the Nightshade poisoning algorithm. Refer to [this repository](https://github.com/saffronrust/fyp_nightshade) for my implementation of Nightshade.

### Step 2: Convert Dataset Format for VLM Training
Prepare the poisoned dataset in the appropriate format for VLM finetuning:
```bash
python VLM_dataset_conversion.py
```

### Step 3: Create Dataset Splits
Organize the dataset into training, validation, and test splits:
```bash
python prepare_split.py --data_json cap.json --images_root /to_be_trained_on_VLM --out_dir splits
```

### Step 4: Fine-tune the VLM
Initiate the VLM finetuning process:
```bash
python train_qwen2vl_lora.py
```

### Step 5: Generate Descriptions from Both Models
Generate text descriptions using the clean and poisoned VLM models:
```bash
python output_clean_VLM.py
python output_poisoned_VLM.py
```

### Step 6: Evaluate Model Performance
Compare and evaluate the performance differences between the clean and poisoned models:
```bash
python evaluate_VLM.py
```

### Step 7: Analyze Results
Parse and analyze the evaluation results from the output JSON file:
```bash
python process_results.py
```
