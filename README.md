# fyp_darksight
An offensive data preprocessing tool to poison and disrupt Vision Language Models using Nightshade. For my final year project.

## Organisation of Repository

```
fyp_darksight/
├── splits/                    # Contains train/validation/test dataset splits
├── test_set/                  # The test dataset used to evaluate the performance of the VLM
├── to_be_trained_on_VLM/      # The dataset used to finetune the target VLM
├── cap.json                   # Caption annotation file of to_be_trained_on_VLM/
├── descriptions_clean.json    # Text descriptions generated by the clean VLM
├── descriptions_poisoned.json # Text descriptions generated by the poisoned VLM
├── evaluate_VLM.py            # Performance evaluation script
├── evaluation_results.json    # Output JSON evaluation file
├── output_clean_VLM.py        # Generate descriptions from clean model
├── output_poisoned_VLM.py     # Generate descriptions from poisoned model
├── prepare_split.py           # Dataset preparation and splitting script
├── process_results.py         # Parse output JSON file
├── README.md                  # This file
├── requirements.txt           # Dependencies to install
├── train_qwen2vl_lora.py      # VLM fine-tuning with LoRA
└── VLM_dataset_conversion.py  # Convert the dataset to be suitable as input to train the VLM 
```

## Setup 
Install dependencies using `pip install -r requirements.txt`
Note that you will also need to install PyTorch, head to [this link](https://pytorch.org/get-started/locally/) to install PyTorch according to your system's requirements.

## Test Dataset Used
[512 x 512 Dogs Images on Kaggle](https://www.kaggle.com/datasets/greg115/dogs-big)

## Convert the dataset to be suitable for finetuning a VLM:
`python VLM_dataset_conversion.py`

## Preparing the dataset to be finetuned:
`python prepare_split.py --data_json cap.json --images_root /to_be_trained_on_VLM --out_dir splits`

## Finetune the VLM:
`python train_qwen2vl_lora.py`

## Prompt the VLMs to output text descriptions based on the test dataset:
`python output_clean_VLM.py`
`python output_poisoned_VLM.py`

## Evaluate the two VLMs' performance:
`python evaluate_VLM.py`

## Parse the output JSON file:
`python process_results.py`
